Introducción a los Servidores de Web

Objetivo principal

Explicar lo que son y cómo funcionan los servidores de web, así como tipos y funciones principales.

Introducción

Un servidor web es una herramienta que permite publicar página web y la creación de entornos interactivos.

Tipos de Servidores de Web

  • Apache
  • Microsoft Internet Server (MIS)
  • Nginx
  • Elasticache
  • Firewal 2

Factores importantes

Algunos factores a considerar al elegir un servidor web incluyen la capacidad, el coste, la escalabilidad, la disponibilidad, y la compatibilidad con otros servicios.

Web Servers en la Nube

La nube ofrece una serie de beneficios para los hosted de servidor web, incluyendo accesibilidad, escalabilidad, seguridad avanzada y un entorno más eficiente para el mantenimiento.

Configuración de un Servidor Web (AWS)

  1. Access Account in AWS.
  2. Create a new Instance in EC2.
  3. Add the necessary Security Groups.

Conectandonos al servidor web

  1. Create access keys for the instance.
  2. Connect to the instance using SSH.
  3. Install the necessary software.

Que es Nginx?

Nginx es un servidor web de código abierto que se utiliza para servir contenido web y aplicaciones.

Instalación de Nginx Linux

                 
                sudo apt update 
                sudo apt install nginx

Iniciar Nginx

                 
                sudo service nginx start 
                sudo service nginx status

Configuración de Nginx

Al instalar tendremos disponible una nueva ruta llamada:

                 
                /etc/nginx/nginx.conf

Para configurar Nginx, se debe editar el archivo de configuración principal, que se encuentra en: 

                 
                /etc/nginx/nginx.conf
