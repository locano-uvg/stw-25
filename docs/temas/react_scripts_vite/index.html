React Scripts

Create React App (Deprecated)

Con el fin de poder utilizar módulos, react crea una primera versión de su propio compilador, que se llama react-scripts.

¿Qué es CRA (Create React App)?

  • Herramienta para iniciar proyectos React
  • Usa Webpack por debajo
  • Lento al iniciar y hacer cambios
  • Difícil de personalizar

¿Qué es react-scripts?

react-scripts es un paquete que contiene todo lo necesario para que react funcione sin la necesidad de utilizar webpack y babel.

webpack es un empaquetador de módulos.

babel es un compilador de javascript.

El empaquetador de módulos es el encargado de empaquetar los módulos de javascript en un solo archivo, para que el navegador pueda entenderlo.

El compilador de javascript es el encargado de compilar el código de javascript de ES6+ en código de javascript de ES5, para que el navegador pueda entenderlo.

Al instalar react-scripts

  • Instala React, React DOM y react-scripts
  • Configura Webpack, Babel y ESLint
  • Crea estructura básica del proyecto

Scripts de react-scripts

  • npm start - Inicia servidor de desarrollo
    • Compila todo el código
    • Habilita Hot Reloading
    • Muestra errores en el navegador
  • npm run build - Crea versión de producción
    • Minifica y optimiza el código
    • Genera archivos estáticos en /build
    • Prepara para despliegue

Para que funcione, se necesita instalar react-scripts en el proyecto. 

mkdir proyecto-stw25-react-scripts
cd proyecto-stw25-react-scripts
npm init -y
npm install react react-dom react-scripts
npm start

Vite

Vite es un nuevo framework de react que se encarga de crear un servidor de desarrollo y un empaquetador de módulos.

Vite es mucho más rápido que react-scripts.

¿Qué es Vite?

  • Herramienta de desarrollo rápida para frontend
  • Usa ES Modules nativos + Rollup
  • Recompila solo lo que cambia
  • Muy fácil de configurar

Scripts de Vite

  • npm run dev - Servidor de desarrollo
    • Usa ES Modules nativos
    • Compilación bajo demanda
    • Hot Module Replacement instantáneo
  • npm run build - Build de producción
    • Usa Rollup para optimización
    • Genera archivos en /dist
    • Mejor rendimiento final

Creando un proyecto con Vite


                npm create vite@latest proyecto-stw25-vite --template react
                cd proyecto-stw25-vite
                npm install
                npm run dev
  • Instala React y Vite
  • Configuración mínima inicial
  • Estructura moderna y limpia
