Introducción a la IA Generativa

Aprenderemos cómo podemos usar inteligencia artificial para generar código automáticamente.

¿Qué es la IA Generativa?

La Inteligencia Artificial Generativa (IAG) es una disciplina que se ocupa de desarrollar sistemas que generen contenido humano, como texto, código o imágenes, con un nivel de calidad similar al humano. Los sistemas de IA Generativa se basan en técnicas de aprendizaje automático, gradients y Redes Neuronales Generativas (GANS).

¿Qué son los LLM?

Los LLM (Lenguaje de Modelos de Lenguaje) son un tipo de modelo de IA Generativa que se especializa en la generación de texto natural. Los LLM son capaces de aprender patrones de lenguaje humano a partir de grandes cantidades de datos y generar texto coherente y relevante en función de estos patrones.

¿Qué es Ollama?

Ollama es una plataforma de desarrollo de código para aplicaciones de IA Generativa que simplifica el proceso de crear, trainear y depurar modelos de IA. Con Ollama, los desarrolladores pueden acceder a una gran variedad de datasets de entrenamiento y herramientas de visualización y monitoreo integral.
Es ideal para iniciantes que quisieran Integrar IA en sus proyectos sin tenir que implementar from ScratchEverything.

¿Cómo instalar Ollama?

Para instalar Ollama, sigue estos pasos:

  • Descarga y instala el Repositorio de Software Offcial para Ollama desde https://ollama.ai/stable/repo.html.
  • Configura tu cuenta de usuario en la GUI de Ollama.
  • Finalmente, verifica la instalación ejecutando los comandos de verificación proporcionados por Ollama.

Validación de instalación

¿Qué es Continue?*

Continue es una extensión de VS Code diseñada para el desarrollo de código con IA Generativa. Con Continue, los desarrolladores pueden acceder a una interfaz integrada para interactuar con los modelos de IA, visualizar su proceso de entrenamiento y hacer predicciones automáticas sobre código o textos basados en sus patrones históricos.

¿Cómo configurar Continue con Ollama?

Para configurar Continue con Ollama, sigue estos pasos:

  • Descarga e instala la extensión de VS Code de Continue desde el Marketplace de Visual Studio.
  • Configura tu cuenta de usuario en la GUI de Ollama.
  • Finalmente, verifica la instalación ejecutando los comandos de verificación proporcionados por Ollama.
