Introducción al HTML

¿Qué es HTML?

El HTML, que significa "Hipertexto mark up language", es un lenguaje de presentación web creado para construir sitios web. Cada página web está compuesta por una serie de elementos que se almacenan en archivos con extensión .html.

Estructura general del HTML

Etiquetas Semánticas

Las etiquetas semánticas son elementos de HTML que se utilizan para estructurar el contenido de una página web. Estas etiquetas permiten a los motores de búsqueda y a los navegadores web interpretar el contenido de una página web de manera más precisa.

Nombre Características
header La primera línea del documento, generalmente un título.
main El contenedor principal que contiene todo el contenido del document.
section Un contenedor de contenido que agrupa elementos relacionados.
article Un contenedor de contenido independiente que puede ser reutilizado.
footer La última línea del documento, generalmente un pie de página.

Conclusión

El HTML es un sistema fundamental para crear sitios web y presentaciones electrónicas. Con sus componentes básicos, se puede estructurar información de manera clara y organizada.
